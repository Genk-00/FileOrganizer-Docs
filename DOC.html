Welcome to File Organizer v1.0.2

This guide will help you turn a messy folder into a clean, organized collection. You can open this guide any time from the app's Help → Open Guide menu.

Quick Start

  1. Click Pick Source to select the messy folder you want to clean up.
  2. (Optional) Click Pick Destination to choose a *different* folder to put the organized files. If you leave this blank, a new _Organized folder will be created inside your Source folder.
  3. Choose your Action and Conflict policy (see details below).
  4. Click Preview to see a summary of what will be moved or copied.
  5. If everything looks good, click Start to begin the operation.

Free vs. Pro Version

By default, File Organizer runs in Free Mode. This mode is perfect for safe, simple sorting. The Pro Version unlocks advanced features for power-users.

How to Upgrade to Pro

You can unlock the Pro features with a lifetime license key tied to your machine.

  1. In the app, go to the File menu and click Upgrade to Pro...
  2. A dialog will appear showing your unique Machine ID (e.g., ABCD-1234).
  3. Send this Machine ID to the developer to purchase your license.
  4. You will receive a License Key back.
  5. Paste your License Key into the same dialog and click "OK".

Pro features will unlock instantly. The license is for this machine and will not be affected if you reinstall the app. However, it will no longer be valid if you reinstall your operating system (like Windows).

Features Explained

Actions

  • Copy (Free): Copies files from the Source to the Destination. The original files are left untouched and safe.
  • Move PRO: Moves files from the Source to the Destination. The original files are deleted. This is the best way to clean a folder.
  • Merge PRO: Behaves like "Copy". It's designed to copy files from multiple messy folders into one, single _Organized destination.

Conflict Policies

This tells the app what to do if a file with the same name already exists in the destination (e.g., image.jpg already exists).

  • Skip (Free): This is the safest option. It will **not** move or copy the new file, leaving the original in place.
  • Rename PRO: This is the default Pro policy. It will move the new file and rename it (e.g., image (1).jpg) so both files are kept.
  • Overwrite PRO: This is a destructive option. It will delete the old file and replace it with the new one. Use with caution!

Filtering PRO

This allows you to organize only specific file types (e.g., only .jpg and .png files).

How Filtering Works:
  • If you leave "Custom output folder name" blank, files will be sorted by their extension (e.g., _Organized/png/, _Organized/jpg/).
  • If you enter a "Custom output folder name" (e.g., "My Holiday Pics"), all filtered files will be put into that one folder (e.g., _Organized/My Holiday Pics/).

Undo Last Move PRO

The "Undo" button will only become active after you perform a Move operation. It uses the session log to find all the files you just moved and attempts to restore them to their original locations.

  • Undo only works for the most recent session.
  • Undo does **not** work for "Copy" or "Merge" (since no originals were deleted).

FAQ & Troubleshooting

Where did my files go?

Your files are safe! By default, the app creates a single, clean folder named _Organized either in your Destination folder (if you chose one) or inside your Source folder (if you didn't). All your sorted categories (Images, Documents, etc.) will be inside that _Organized folder.

Where are the settings and logs?

  • Settings: Your app settings (like your last-used folders) are saved in your Windows user profile. The path is: C:\Users\YourName\AppData\Roaming\FileOrganizer\settings.json
  • Logs: A detailed log of every operation (move, copy, error) is saved inside your _Organized folder, in a subfolder named .logs/.

My License Key doesn't work!

Please double-check that the key you entered is an exact match for the one you received. Also, keys are tied to the Machine ID. If you are trying to use a key on a different computer, it will not work.
